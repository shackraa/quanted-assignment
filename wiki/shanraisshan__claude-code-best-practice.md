## Overview
A continuously updated reference repo cataloguing Claude Code features, best practices, and community workflows — subagents, commands, skills, hooks, MCP servers, plugins, settings, and dozens of "development workflow" patterns other teams have published, each broken into its command → agent → skill pipeline.

## Problem It Solves
Claude Code's feature surface grows quickly and documentation is spread across many sources; this repo functions as a single, comparison-friendly map of what exists, whether it's implemented with an example, and how other public workflows have composed the primitives (agents, commands, skills) into pipelines.

## Key Features / Use Cases
- **Concept-to-implementation table** — every major Claude Code feature (subagents, commands, skills, hooks, MCP, plugins, memory, etc.) linked to a best-practice doc and a working implementation example.
- **Development workflow comparisons** — a table of public multi-step workflows (Superpowers, Spec Kit, BMAD-METHOD, and others) showing their step sequence and how many agents/commands/skills each uses.
- **"Hot" features tracker** — newer/beta Claude Code capabilities (Ultrareview, Devcontainers, Channels, Agent Teams, Scheduled Tasks) tracked as they ship.
- **Cross-model workflow guidance** — patterns for using Claude Code alongside Codex, Gemini, and other models via plugins, MCP, or router swaps.
