## Overview
A free, structured Microsoft course teaching the fundamentals of building AI agents, with code samples built on Microsoft Agent Framework and Azure AI Foundry. Translated into 50+ languages.

## Problem It Solves
"AI agent" concepts (planning, tool use, RAG, multi-agent design, trustworthiness, protocols like MCP/A2A) are scattered across docs and papers; this course sequences them into a coherent, hands-on curriculum for someone starting from zero.

## Key Features / Use Cases
- **Lesson-by-lesson structure** — from "intro to agents" through tool use, planning, multi-agent design, metacognition, production deployment, and security.
- **Code samples per lesson** — runnable Python examples for each design pattern, not just theory.
- **Companion video** for most lessons.
- **Covers agentic protocols** — MCP, A2A, and NLWeb get a dedicated lesson, plus one on context engineering.
- **Part of a course family** — links to Microsoft's other "for Beginners" series (Generative AI, MCP, ML, etc.).
