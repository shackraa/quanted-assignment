## Overview
A long-running, comprehensive reference on prompt engineering — techniques, papers, courses, and a companion [web version](https://www.promptingguide.ai/) — maintained by DAIR.AI. It's a learning resource rather than a piece of software.

## Problem It Solves
Getting reliable behavior out of LLMs is a skill with real technique behind it (few-shot examples, chain-of-thought, ReAct, self-consistency, etc.), and this guide collects that technique in one place instead of scattering it across blog posts and papers.

## Key Features / Use Cases
- **Technique catalog** — zero-shot, few-shot, chain-of-thought, tree-of-thoughts, ReAct, automatic prompt engineering, and more, each with explanation and examples.
- **Applications section** — function calling, synthetic data generation, code generation, classification.
- **Risks & misuses** — adversarial prompting, factuality, and bias sections, not just capability-boosting techniques.
- **Model-specific notes** — guidance tuned to specific model families (GPT-4, LLaMA, Mistral, Gemini, etc.).
- **Courses** — a paid DAIR.AI Academy track that goes deeper than the free guide, including AI Agents content.
