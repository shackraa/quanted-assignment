# Awesome Claude Skills

**Stars:** 66,666
**Topics:** agent-skills, ai-agents, automation, claude, claude-code, mcp

## Overview
A curated, categorized directory of 1000+ Claude Skills and plugins — reusable instruction packages (a `SKILL.md` with frontmatter plus optional scripts/references) that teach an agent how to handle a specific class of task, spanning Claude.ai, Claude Code, and other agents that adopted the open Skills standard (Codex, Cursor, Gemini CLI, Antigravity, Windsurf).

## Problem It Solves
As the Skills ecosystem grew past Anthropic's own bundled set, there was no single place to discover what the community had built. This list solves discovery, organized by use case rather than alphabetically.

## Key Features / Use Cases
- **Category browsing** — document processing, dev tools, data/analysis, business, communication, creative, productivity, collaboration, security, and app automation via Composio.
- **Explains the format itself** — a short primer distinguishing skills (workflow/behavior) from MCP (connections) and tools (actions), useful context for anyone new to the ecosystem.
- **Connect-apps plugin** — a bundled skill that lets Claude take real actions (send email, create issues, post to Slack) across 500+ apps via Composio.
- **Community-sourced** — most entries link to external repos maintained by individual contributors, with attribution.
