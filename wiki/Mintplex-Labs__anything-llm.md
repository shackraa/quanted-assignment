## Overview
AnythingLLM is an all-in-one, self-hostable AI application: chat with your own documents, run built-in AI agents, and connect to nearly any LLM/embedder/vector-database provider — closed or open-source — without needing to assemble a RAG + agent stack from separate pieces.

## Problem It Solves
Standing up a private "ChatGPT over my documents" experience normally means wiring together a model provider, an embedding model, a vector store, and an agent layer yourself. AnythingLLM packages all of that into one multi-user application that runs locally by default.

## Key Features / Use Cases
- **Document chat** — PDF, TXT, DOCX and more, with source citations.
- **Built-in AI agents** — web browsing and other agent capabilities inside a workspace, plus a no-code agent-flow builder.
- **Dynamic model routing & memory** — routes chats to the best configured model, and can remember information across sessions per user/workspace.
- **Broad provider support** — dozens of LLM, embedder, and vector-database backends, including fully local options (Ollama, LM Studio, LocalAI).
- **Multi-user & MCP-compatible** — permissioned multi-user instances, plus MCP support for external tool integrations.
